Setup Prerequisites

API Keys Setup

  1. OpenAI API Key

    Get your API key from OpenAI Platform

  2. SerperDev API Key

    Get your API key from SerperDev

Installation & Setup

  1. Install crewAI
    pip install crewai
  2. Environment Variables
    export OPENAI_API_KEY='your-openai-key' export SERPER_API_KEY='your-serper-key'

    For Windows, use:

    set OPENAI_API_KEY=your-openai-key set SERPER_API_KEY=your-serper-key
