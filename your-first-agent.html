Your First Agent

Task Overview

In this assignment, you'll create your first AI agent using Python. You'll learn how to set up the basic structure, implement core functionality, and make your agent interact with users.

Code Example


import openai

class Agent:
    def __init__(self, api_key):
        self.api_key = api_key
        openai.api_key = api_key

    def process_input(self, user_input):
        # Your agent's logic here
        response = openai.ChatCompletion.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": user_input}]
        )
        return response.choices[0].message.content

# Example usage
if __name__ == "__main__":
    agent = Agent("your-api-key-here")
    response = agent.process_input("Hello, agent!")
    print(response)
